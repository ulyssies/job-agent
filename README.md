# Job Match Agent 🤖

A personal AI-powered job matching agent built with Node.js and Claude AI.

## What it does
- Searches multiple job boards (Adzuna, The Muse, Jobicy) across target cities daily
- Uses Claude AI to score each listing against my resume (0-100% match)
- Generates a live dashboard with match scores, salary estimates, and apply links
- Sends a daily email summary with top matches per city

## Built with
- Node.js
- Anthropic Claude API (claude-sonnet-4-20250514)
- Adzuna Jobs API
- Nodemailer
- Vanilla HTML/CSS/JS dashboard
