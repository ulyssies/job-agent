🤖 Job Match Agent

A personal AI-powered job matching agent built with Node.js and Claude AI. Searches multiple job boards daily, scores listings against my resume, estimates salaries, and delivers ranked results to my inbox every morning.

View Live Dashboard →

What It Does

  • Searches multiple job boards (Adzuna, The Muse, Jobicy) across 6 target cities daily — Atlanta, Austin, Seattle, Denver, Chicago, and Remote
  • Uses Claude AI to parse my resume and score each listing 0–100% based on skill and experience match
  • Estimates salary ranges using AI when job boards don't list compensation
  • Deduplicates results across runs so only fresh listings appear each day
  • Generates a persistent dark-mode dashboard tabbed by city with sortable columns, search filtering, and 30-day run history
  • Sends a daily HTML email summary with top matches per city including match %, salary estimate, and one-line fit reason

Built With

Node.js (ES Modules) Anthropic Claude API claude-sonnet-4-20250514 Adzuna Jobs API The Muse API Jobicy API Nodemailer + Gmail Vanilla HTML/CSS/JS

Project Structure

job-agent/
  ├── config.js       // target cities and job titles
  ├── main.js        // master orchestrator
  ├── resume.txt     // resume (gitignored)
  ├── src/
  │   ├── parseResume.js  // AI resume parsing
  │   ├── jobSearch.js    // multi-source job fetching
  │   ├── scoreJobs.js    // AI scoring + salary estimation
  │   ├── database.js     // persistent run history (30 days)
  │   ├── dashboard.js    // HTML dashboard generation
  │   └── emailSummary.js // daily email report
  └── data/
      └── jobs.json     // auto-generated run history

Setup

npm install
cp .env.example .env
# Fill in your API keys in .env
# Paste your resume into resume.txt
node main.js

Automating Daily Runs

# Mac/Linux — runs every day at 7am
crontab -e
0 7 * * * cd /path/to/job-agent && node main.js

Windows: Use Task Scheduler to run node main.js daily.
